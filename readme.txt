# Text Classification for Support Tickets

## Description
This project classifies customer support tickets into predefined categories using NLP and Machine Learning.

## Technologies
- Python
- NLTK
- Scikit-learn
- TF-IDF
- Logistic Regression

## Steps to Run
1. Install dependencies
   pip install -r requirements.txt

2. Run preprocessing
   python src/preprocess.py

3. Train model
   python src/train_model.py

4. Evaluate model
   python src/evaluate_model.py

## Output
- Classification report
- Confusion matrix visualization
- Saved ML model and vectorizer
